# Archived InvestorStack Aesthetic

This folder preserves the pre-overhaul InvestorStack visual system before the Humanist Compute Atelier rebuild.

## Snapshot

- Last pre-overhaul commit: `e58c7c8c671a90a5e64a2423a2424d44fc81f6b1`
- Captured: 2026-06-24
- Live direction at capture: institutional pastoral interface with warm paper surfaces, orchard imagery, dark-mode support and a visible theme toggle.
- New direction: Humanist Compute Atelier, a single canonical archival-paper operating-stack atlas with no exposed light/dark toggle.

## Contents

- `theme-notes.md`: what the archived aesthetic was.
- `restoration-notes.md`: how to restore or reintroduce it.
- `component-map.md`: where the theme lived in the live app.
- `tokens-snapshot.css`: token snapshot from `src/styles/tokens.css`.
- `css-or-theme-snapshot.css`: global CSS snapshot from `src/app/globals.css`.
- `screenshots/`: desktop, Stack Builder and mobile verification screenshots from the previous pass.

This archive is documentary. The live app should not import files from here.
